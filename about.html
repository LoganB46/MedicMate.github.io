Medic Mate

About us

Hello, Medicmate is a personal project I'm working on to bring important tools to the fingertips of emergency healthcare providers when they need it most. There will be bugs to this website, that will be taken care of as time passes. I plan on adding more tools over time as I think of them or they are requested by anyone.
If you have any suggestions, feedback, or found any bugs, please leave them in the GitHub Issues section of the project.
If you would like to contribute to the project, please feel free to fork the repository and submit a pull request with your changes. GitHub Repository

Current Features and known issues

Features:

  • Burn Calculator: Calculates the total body surface area (TBSA) affected by burns using the Rule of Nines.
  • Drug Calculators: Calculates drug dosages based on patient weight. Drip rate calculator.
  • EMSA Score Calculator
  • GCS (Glascow Coma Scale) Calculator.
  • Hospital phone directory that will show correct contact information at a later time.
  • Ideal Body Weight Calculator.
  • Diversion page with admin login and User interface page.
  • Protocols page that supports PDF protocols specific for your service. With an indexing system that allows the user to navigate the PDF quickly when needed.

Known Issues:

  • Admin diversion manegment page does not function at this time.
  • Login system is not secure it is currently a simple functioning placeholder during development.
  • Burn calculator does not highlight the affected body parts correctly, although when you click on the body part it should still register with the site and still calculate the TBSA.
